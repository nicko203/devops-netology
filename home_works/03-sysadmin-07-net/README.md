### 1. Проверьте список доступных сетевых интерфейсов на вашем компьютере. Какие команды есть для этого в Linux и в Windows?  

Команды для вывода списка сетевых интерфесов:  
- Linux - ifconfig, ip a  
- Windows - ipconfig
